# 표(table)

* 테이블(표)를 생성합니다.
* 파이프( \| )와 하이픈( \- )을 이용해서 행과 열을 구분합니다.
* 혹은 본문 >> 표 >> 표삽입

* 예시 \|동물|종류|다리 개수|

| 동물 | 종류   | 다리 개수 |
| ---- | ------ | --------- |
| 사자 | 포유류 | 4         |
| 닭   | 조류   | 2         |



수평선

* 구분 선을 생성합니다.
* \- 또는  \* 을 **3**개 이상 연속으로 작성합니다.
